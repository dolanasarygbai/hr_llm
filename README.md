# HR-Аналитик на основе локальной LLM

Проект для анализа соответствия резюме вакансиям с использованием локальной модели gemma3:1b через Ollama.

## Установка

1. Установите Ollama с официального сайта: https://ollama.com/
2. Скачайте легковесную модель: `ollama pull gemma3:1b`
3. Рекомендую поставить виртуальное окружение.
4. Установите зависимости: `pip install -r requirements.txt`

## Использование

Поместите файлы с именами, содержащими "vacancy" и "resume", в папку с проектом и запустите:
```bash
python main.py
