# hr_llm
- LLM оценивает насколько резюме кандидата подходит под вакансию. Инструмент может быть помощником для HR. 

- Чтобы запустить инструмент, введите в командной строке python FILE_NAME.PY vacance.txt resume.pdf

- Я создавала этот инструмент с помощью vibe coding.
  
- Использована легковесная модель gemma3:1b на локальном сервере ollama. Поэтому установите Ollama с официального сайта. Затем введите команду ollama pull gemma3:1b в командной строке.

- Рекомендую поставить виртуальное окружение. Файл с зависимостями requirements.txt.
